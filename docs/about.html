About

Productivity Tracker Capstone — CMPA4404

What is this?

Productivity Tracker is a full-stack habit and task management web application built as a capstone project. It helps students and young adults stay consistent with daily habits, manage tasks by category, and plan ahead using a built-in calendar — all without the complexity of traditional productivity apps.

Key Features

Tasks are stored in a backend database so they persist across devices and browser sessions. The calendar view lets you schedule tasks to specific dates and see your workload at a glance. Daily habits track consecutive-day streaks to keep you motivated. Dark mode and category color-coding make the experience comfortable and organized.

Backend Storage Calendar Planning Habit Streaks Dark Mode Category Filtering Progress Tracking

Tech Stack

HTML / CSS / JavaScript Node.js Express SQLite (better-sqlite3) REST API

Why this approach?

Existing tools like Habitica are engaging but complex; Loop Habit Tracker lacks a web experience; Streaks is Apple-only. This project balances simplicity and engagement in a web-based format accessible to anyone, while adding a deliberate planning layer through the calendar and cross-device persistence through the backend.
